## What I Learned

- `var` scope needed for Playwright E2E test access
- Web Audio API procedural sounds avoid external file dependencies
- Combo systems need visible feedback (popups + timer bar)
- Screen shake with decaying intensity adds great game feel
- Deterministic test setups (force-spawn, teleport) are more reliable than playing through

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| Web Audio API for sounds | No files to serve, works everywhere |
| SVG favicon | Scalable, tiny file size, matches game visuals |
| Combo thresholds at 3/6/10/15 | Gradual difficulty, achievable in normal play |
| Screen shake 12 intensity, 20 frames | Noticeable but not nauseating |
